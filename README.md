## Setup

First, make sure you have [node and npm installed](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

Then, install the dependencies:

`npm install`

Once installed, you can start the development server:

`npm run dev`

This will start the development server at `http://localhost:5173`.

## Build

To build the project assets, run:

`npm run build`
